What is a Linked List?

A linked list is a linear data structure where each element is a separate object. Each element (node) of a list contains two items: the data and a reference to the next node.

Features of Linked Lists

  • Dynamic size
  • Efficient insertions and deletions
  • Can easily grow and shrink

Linked List Example

struct Node {
    int data;
    struct Node* next;
};
