What is a Stack?

A stack is a linear data structure which follows the Last In, First Out (LIFO) principle. Elements are added to and removed from the same end, called the top of the stack.

Features of Stack

  • Last In, First Out (LIFO)
  • Push and Pop operations
  • Efficient for certain algorithmic tasks

Stack Example

stack.push(10);
stack.push(20);
stack.pop();
